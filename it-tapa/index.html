    IT-Tapa middagsessie

    • IT-Tapa op Facebook
    • GIP website Bryce van den Brande

    Tijdens de IT-Tapa middagsessie hebben Bryce en ik het gehad over Wiggio. Wiggio is een tool die leerkrachten kunnen gebruiken in hun lessen. Als je een groep hebt aangemaakt, kan je er heel wat zaken mee doen zoals bestanden delen, links delen van bijvoorbeeld website’s met oefeningen, een poll maken, een standaard tekstberichtje plaatsen, meetings aanmaken, events aanmaken, een to-do-lijstje opstellen,… Met andere woorden kan wiggio wel eens handig zijn tijdens de lessen.

    Om terug een keuze te kunnen maken over welke taak u wil bekijken, kan u links in het menu klikken op ‘IT-Tapa’
